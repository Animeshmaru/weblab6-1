TASK-1 :

Create an object ‘Person’ in Java Script, which will have ‘FirstName’, ’LastName’ and ‘Office Address’ as properties within it. Create three Persons and print all those three properties.

ANS :

Creating an object and it's properties :

JavaScript Code

  
    <script> 

        function Person(fname, lname, address) 
        {
            this.firstName = fname;
            this.lastName = lname;
            this.officeAddress = address;
        }

        const personOne = new Person('Animesh', 'Maru', 'Bhilwara, Rajasthan');
        const personTwo = new Person('Himanshu', 'Rai', 'Patna, Bihar');
        const personThree = new Person('Rachit', 'Suneja', 'Kota, Rajasthan');
            
        console.log("Person 1 : ");
        console.log("First Name: "+ personOne.firstName )
        console.log("Last Name: "+ personOne.lastName )
        console.log("Office Address: "+ personOne.officeAddress )
                
        console.log("Person 2 : ");
        console.log("First Name: "+ personTwo.firstName )
        console.log("Last Name: "+ personTwo.lastName )
        console.log("Office Address: "+ personTwo.officeAddress )
                
        console.log("Person 3 : ");
        console.log("First Name: "+ personThree.firstName )
        console.log("Last Name: "+ personThree.lastName )
        console.log("Office Address: "+ personThree.officeAddress )

    </script>

Output
